# sao-workspace

> generator for npm workspace with linting, formating, lerna...

## Usage

Install [SAO](https://github.com/saojs/sao) first.

```bash
yarn global add sao
# or
npm i -g sao
```

### From npm

```bash
sao workspace my-project
```

### From git

```bash
sao amirilovic/sao-workspace my-project
```
